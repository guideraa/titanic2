Action Select Columns and Clean Data

Needs:

  • URL for data source. This can be a web URL or a local file name
  • All of the column names in the data source.

Creates:

  • Selected columns. User chooses which columns of the available columns will be included in the model.
  • Once the user has selected the columns to use in the model, those columns are serialized and stored in a file called "included_columns.pkl"
  • Once the DataFrame for the model is created with One Hot Encoding (OHE), the column names are serialized and stored in a file called "model_columns.pkl".
  • DataFrame with specified included columns together with One Hot Encoded (OHE) columns for any non numerical columns. Therefore with OHE, all columns in the model are numeric. Also, missing values in the DataFrame are replaced with zeros.

Action Build and Train Model

Needs:

  • A OHE pandas DataFrame that has been cleaned.
  • Name of data column that will be the model's label (string).

Creates:

  • Trained model: sklearn.linear_model.LogisticRegression Object
  • List (strings) of model features

Action Make Prediction

Needs:

  • Trained model: sklearn.linear_model.LogisticRegression Object
  • Test data in dictionary form

Creates:

Prediction as an ndarray
